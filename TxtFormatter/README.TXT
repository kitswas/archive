PROJECT TITLE: TxtFormatter

PURPOSE OF PROJECT: This command-line application creates a formatted text file (required by a certain bank) 
from the text file (in another format) provided by the user. 

VERSION or DATE: 20/03/2019

AUTHORS: Swastik Pal

USER INSTRUCTIONS: The end product delivered to you should be the file TxtFormatter.jar. 
Use command "java -jar Tax.jar" to execute. Java should be installed on your machine.
